Web design refers to the design of websites that are displayed on the internet.It usually refers to the user experience aspects of website development rather than software development. A web designer works on the appearance, layout and in some cases, content of website.

What web-designers do?

Web design identifies the goals of a website webpage and promotes accessibility for all potential users. This process involves organizing content and images across series of pages, integrating applications and other interactive elememts.


The professionals who perform this process are callled web designers, and their job includes the following duties:
  • Selecting easy-to-read fonts.
  • Choosing attractive color schemes that also easy-to-read fonts.
  • Implementing a brand's identity into the colors, fonts and layout.
  • Creating a map of website's structure to ensure intuitive navigation.
  • Placing images, logos, text, videos, applications and other elements.
  • Using coding languages, such as HTML and CSS, to create layouts and to style pages.
  • Making optimized versions of websites and pages both for desktop and mobile viewing.
